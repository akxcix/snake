# Snake
Snake game implemented in C using Allegro

## Building from source

### Requirements
* Allegro5: `pacman -S allegro`
* Make: `pacman -S make`
* CMake: `pacman -S cmake`

### Compiling
```
git clone https://www.github.com/iamadarshk/snake
cd snake
cmake .
make
```
### Running
```
./snake_game
```
